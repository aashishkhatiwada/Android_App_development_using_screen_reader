pre requirements before you get started

Developing android apps is an easy and enjoyable process, however, without having certain technical knowledge, it is impossible to accomplish that.
Inorder to facilitate blind developers, we are listing out some essential skills which 1should learn before actually getting into the business
Keep reading below:

A programming language(java):

  • basics of programming including datatypes, variables, operators, keywords, conditional statements, loops, switch statements, arrays, methods, and string manipulation.
  • object oriented programming including classes, objects, inheritence, polymorfism, object construction, member visibility, abstraction, interfaces, exception handling etc.
  • java collections including lists, maps and sets
  • concurrency including basics of threads, runnables, interThread communication, etc.
  • regular expressions.
  • recurssion.
  • file handling.

In addition to the above requirements, knowing some basics of xml is also reccomended.
